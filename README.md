# Проект-тренировка

Просто хотелоось создать свой сайт, получилось вот так.

В проекте используетс React, для сборки используется webpack 4, для деплоя на ftp используется gulp.

# Работа с проектом

## Начало работы с проектом

```
npm install
```
и
```
npm start
```

## Сбилдить проект

```
npm run build
```

## Выгрузка на ftp

```
npm run deploy
```
